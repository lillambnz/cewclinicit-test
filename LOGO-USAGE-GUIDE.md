# ClinicIT.Solutions Logo Usage Guide

## 🎨 Logo Variations

### 1. **Primary Logo** (`logo.svg`)
- **Use for**: Main branding, website headers, business cards, letterheads
- **Background**: Light backgrounds (white, light grey)
- **Dimensions**: 400x120px (scalable SVG)

### 2. **Icon Only** (`logo-icon.svg`)
- **Use for**: Social media profiles, app icons, favicons, small spaces
- **Background**: Any background (has gradient background)
- **Dimensions**: 100x100px (scalable SVG)

### 3. **Stacked Version** (`logo-stacked.svg`)
- **Use for**: Vertical layouts, business cards, mobile apps
- **Background**: Light backgrounds
- **Dimensions**: 300x200px (scalable SVG)

### 4. **White Version** (`logo-white.svg`)
- **Use for**: Dark backgrounds, overlays, presentations
- **Background**: Dark backgrounds only
- **Dimensions**: 400x120px (scalable SVG)

### 5. **Favicon** (`favicon.svg`)
- **Use for**: Browser favicons, small web icons
- **Background**: Transparent or any color
- **Dimensions**: 32x32px (scalable SVG)

## 🎯 Usage Guidelines

### ✅ **DO:**
- Use on clean, uncluttered backgrounds
- Maintain minimum spacing around logo (equal to icon height)
- Scale proportionally (don't stretch or distort)
- Use high contrast between logo and background
- Use provided color versions for brand consistency

### ❌ **DON'T:**
- Place on busy or patterned backgrounds
- Change colors unless using approved white version
- Rotate or skew the logo
- Use low resolution versions for print
- Place text too close to the logo

## 📐 Spacing & Sizing

### **Minimum Clear Space:**
- All sides: Equal to the height of the icon (circular element)
- Never place other elements within this space

### **Minimum Sizes:**
- **Digital**: 120px wide (primary logo)
- **Print**: 1 inch wide (primary logo)
- **Favicon**: 16px minimum

## 📱 Digital Applications

### **Website:**
- Header: Primary logo (logo.svg)
- Favicon: favicon.svg
- Social sharing: logo-icon.svg

### **Social Media:**
- **Profile Pictures**: logo-icon.svg (Instagram, Facebook, LinkedIn)
- **Cover Photos**: logo.svg or logo-stacked.svg
- **Posts**: Any version depending on layout

### **Email Signatures:**
- Use: logo.svg
- Max width: 200px
- Link to website

## 🖨️ Print Applications

### **Business Cards:**
- Use: logo-stacked.svg or logo.svg
- Minimum: 1 inch wide

### **Letterheads:**
- Use: logo.svg
- Position: Top left or center
- Size: 2-3 inches wide

### **Brochures/Flyers:**
- Use: Any version depending on layout
- Ensure high resolution for print

## 📁 File Formats Available

### **Vector (Recommended):**
- `.svg` - Scalable Vector Graphics (web, print, any size)

### **Additional Formats Available on Request:**
- `.png` - Transparent background (web, presentations)
- `.jpg` - Solid background (general use)
- `.pdf` - Print ready (professional printing)
- `.eps` - Vector (professional design software)
